HTML

  • HTML is used to create the structure of a webpage. It is made up of elements which work together to provide content and basic framework for all webpages.
  • HTML elements are usually composed of an opening and closing tag represented with angle brackets.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • CSS stands for Cascading Style Sheets
  • Think of it as the paint and decorations of the house.
  • You can use it to declare fonts, colors, and other style assets.
  • There are three different ways to style in CSS: Inline, Internal, and External.

Git

  • Git - Software on your computer that tracks changes you make to your code.
  • This is how you can code using Version Control (creates a SHA, which is a long string of numbers representing that version of the code)
  • You can use git by using git commands
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • GitHub is a web interface for Git

JavaScript

  • JavaScript is a language used to add functionality to a website
  • Think of it as the ability to interact with the house (opening doors, running water)
